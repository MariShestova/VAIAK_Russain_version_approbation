# VAIAK_Russain_version_approbation
Psychometric validation of the Russian version of the Vienna Art Interest and Art Knowledge Questionnaire (VAIAK). Includes EFA, CFA, Omega, MGCFA, multi-group IRT (GRM &amp; 2PL), DIF analysis with ESSD effect size, and group comparisons (Novices vs. Experts). Full reproducibility of all tables from the manuscript. R code.
